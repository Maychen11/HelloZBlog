# Z-BlogPHP 插件开发演示

Z-BlogPHP 插件开发演示

## 开始

请在本地部署 Z-BlogPHP 环境；

将本项目文件夹放置在`zb_users\plugin`内并重命名为 HelloZBlog；

进入插件管理启用本插件；

安装 VSCode，之后双击插件内的`HelloZBlog.code-workspace`；

配合[**教程文档**](docs#readme "教程文档")，开始折腾吧。。
